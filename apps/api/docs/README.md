# API Documentation

This is the Nextra-based documentation site for the Nor Chain Explorer API.

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Structure

- `pages/` - Documentation pages (MDX format)
- `theme.config.tsx` - Theme configuration
- `next.config.js` - Next.js configuration

## Adding Documentation

1. Add new pages to `pages/` directory
2. Update `pages/_meta.json` to include new pages
3. Use MDX format for rich content

## Features

- ✅ Search functionality
- ✅ Dark mode support
