# API Overview

The NorChain Explorer API provides comprehensive access to blockchain data through REST and GraphQL interfaces.

## Base URLs

### Development
```
http://localhost:3000/api/v1
```

### Production
```
https://api.norchain.org/api/v1
```

## API Versions

The API uses URI versioning:
- **v1**: Current stable version (`/api/v1/*`)

## Response Format

All REST API responses follow this format:

```json
{
  "status": "1",
  "message": "OK",
  "result": {
    // Response data
  }
}
```

### Error Response

```json
{
  "status": "0",
  "message": "Error description",
  "result": null
}
```

## Authentication

The API supports two authentication methods:

### 1. JWT Tokens

Include in `Authorization` header:
```
Authorization: Bearer <token>
```

### 2. API Keys

Include in `X-API-Key` header:
```
X-API-Key: nk_abc123...
```

See [Authentication](/api/authentication) for details.

## Rate Limits

- **Unauthenticated**: 100 requests/minute
- **Authenticated**: 1000 requests/minute
- **Premium**: Custom limits

Rate limit headers are included in responses:
```
X-RateLimit-Limit: 1000
X-RateLimit-Remaining: 999
X-RateLimit-Reset: 1640995200
```

## Endpoints

### Account Endpoints
- `GET /api/v1/account/balance` - Get account balance
- `GET /api/v1/account/txlist` - Get transaction list
- `GET /api/v1/account/tokenlist` - Get token list
- `GET /api/v1/account/tokentx` - Get token transfers
- `GET /api/v1/account/balancemulti` - Multi-address balance
- `GET /api/v1/account/txlistinternal` - Internal transactions
- `GET /api/v1/account/summary` - Account summary

### Block Endpoints
- `GET /api/v1/block/getblock` - Get block information
- `GET /api/v1/block/getblockreward` - Get block reward
- `GET /api/v1/block/getblockcountdown` - Get countdown
- `GET /api/v1/block/getblocknumber` - Latest block number

### Transaction Endpoints
- `GET /api/v1/transaction/gettxreceiptstatus` - Receipt status
- `GET /api/v1/transaction/getstatus` - Transaction status
- `GET /api/v1/transaction/gettxinfo` - Transaction details

### Token Endpoints
- `GET /api/v1/token/tokensupply` - Token supply
- `GET /api/v1/token/tokenaccountbalance` - Token balance
- `GET /api/v1/token/tokeninfo` - Token info
- `GET /api/v1/token/tokentx` - Token transfers

### Contract Endpoints
- `GET /api/v1/contract/getabi` - Get ABI
- `GET /api/v1/contract/getsourcecode` - Get source code
- `POST /api/v1/contract/verifycontract` - Verify contract

### Stats Endpoints
- `GET /api/v1/stats/ethsupply` - ETH supply
- `GET /api/v1/stats/ethprice` - ETH price
- `GET /api/v1/stats/chainsize` - Chain size
- `GET /api/v1/stats/nodecount` - Node count

### Advanced Endpoints
- `POST /api/v1/batch/balances` - Batch balances
- `POST /api/v1/batch/transaction-counts` - Batch transaction counts
- `GET /api/v1/analytics/portfolio` - Portfolio analytics
- `GET /api/v1/gas/gasoracle` - Gas oracle

## GraphQL API

The API also provides a GraphQL endpoint:

```
POST http://localhost:3000/graphql
```

See [GraphQL API](/api/graphql) for details.

## WebSocket Support

Real-time updates via WebSocket:

```
ws://localhost:3000
```

## Swagger Documentation

Interactive API documentation available at:
```
http://localhost:3000/api-docs
```

## SDKs and Libraries

### JavaScript/TypeScript

```typescript
import { ExplorerAPI } from '@norchain/api-client';

const api = new ExplorerAPI({
  baseUrl: 'http://localhost:3000/api/v1',
  apiKey: 'your-api-key'
});

const balance = await api.account.getBalance('0x...');
```
