# NorChain Documentation

Comprehensive documentation for the NorChain ecosystem built with Nextra.

## Overview

This documentation site provides complete documentation for:
- Explorer API
- Explorer App
- Landing Page
- NEX Exchange
- Development guides
- API reference

## Development

```bash
npm install
npm run dev
```

The documentation will be available at http://localhost:3011

## Building

```bash
npm run build
npm run start
```

## Structure

- `pages/` - Documentation pages (MDX format)
- `theme.config.tsx` - Theme configuration
- `next.config.mjs` - Next.js configuration

## Adding Documentation

1. Create a new `.mdx` file in `pages/`
2. Update `pages/_meta.json` to add it to the navigation
3. Write your documentation using MDX syntax

## Features

- ✅ Search functionality
- ✅ Dark mode support
- ✅ Responsive design
- ✅ Code syntax highlighting
- ✅ Automatic navigation
- ✅ Edit links to GitHub
