# Desktop Wallet - Nor Wallet

## Setup

### Prerequisites

- Rust (latest stable)
- Node.js (18+)
- npm or yarn

### Installation

```bash
cd desktop-wallet
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Features

- ✅ Rust backend (Tauri)
- ✅ Supabase integration
- ✅ Wallet management
- ✅ Cross-platform sync
- ✅ Glassmorphism UI

## Structure

- `src/` - Frontend (HTML/JS)
- `src-tauri/src/` - Rust backend
- `src-tauri/src/services/` - Supabase services
